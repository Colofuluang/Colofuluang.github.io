C++/Linux 后台服务器开发高级架构师学习路线

2024/9/8 学习路线 架构师 Linux C++

大四学习路线。

0. 前言

上网搜了一下腾讯校招的关于软件开发和测试开发岗位实习的要求，发现里面Linux后台开发是属于加分项，想来自己没有只是会用Linux系统的基本操作，这是远远不够的，因此去网上开始自己的Linux学习之路。

1. Linux 开发环境

  1. 了解 Linux 环境搭建，了解 LinuxC 编程
  2. 了解 Linux 安装，命令使用，shell 编程
  3. shell 脚本实现检测局域网内哪些ip地址宕机

2. LinuxC 编程

  1. LinuxC 编程，统计文件单词数量，包括文件操作和文件指针
  2. LinuxC 编程，实现通讯录，包括通讯录等

3. Linux 环境编程

  1. 并发下的计数方案，包括互斥锁、自旋锁、原子操作等
  2. 实现线程池，包括如线程队列，任务队列和条件变量等
