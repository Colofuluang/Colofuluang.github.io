小绵羊和粥粥的日常

2024/10/23 日记 恋爱

我想故事里面的她，无论在什么地方，只要回头，都有人在。

2024.10.23

  1. 今天开始第一次写日记
  2. 我看书，她上课，我爱她，她也爱我
  3. 粥粥和我抱怨英语课上的逆天组员，这样的男生真的很烦人的，粥粥一节课都起来3次了都，可恶可恶可恶
  4. 看书看到婚戒了，我想一周年的时候把我们的银戒指换成金戒指，然后订婚的时候换成金钻，结婚的时候换成钻戒，真美好
  5. 我一向吃饭吃的不多，今天吃自选，居然吃了34块，还吃了俩碗饭，粥粥夸我棒，我迟早变壮的
  6. 看《有的是时间》感觉很多瞬间很温暖，高亮标注了，希望粥粥喜欢

2024.10.24

  1. 粥粥和我十点半才起床，嘿嘿嘿，睡的很香
  2. 我在织围巾，给粥粥看了半成品，粥粥说我好厉害
  3. 打算去吃饭了，和粥粥说一声
  4. 就喜欢粥粥给我转发抖音，喊我宝宝、老公、乖乖嘿嘿嘿

2024.10.25

  1. 又惹粥粥不开心了，今天带着情绪睡觉了

2024.10.26

  1. 大晚上突然说要去吃海底捞，说去就去，吃到凌晨两点半
  2. 其实早上粥粥很着急，还哭了，她说见不到我，错过了大巴车，没事的啦
  3. 深圳真没什么好玩的呜呜呜

2024.10.27

  1. 躺到下午两点退房，出来两个人不知道去哪里，陪粥粥去洗头发，因为姨妈来了，洗不了
  2. 本来今天就要回广州了，粥粥又留深圳玩了一天，嘿嘿嘿，晚上可以去吃烧烤了

2024.10.28

  1. 下午看电影去啦，偷偷带了炸鸡和薯条，被保安狠狠拿下
  2. 中午点的麦当劳，还是挺好吃的
  3. 看完回学校，在山上坐会，好冷呜呜呜
  4. 送粥粥回去，粥粥说很想我，要把我身上的气味记住

2024.10.29

  1. 今天体检去了，我就知道，尿检不合格，又去复查了下午，以后一定多多喝水
  2. 吃了俩老北京鸡肉卷，真的好吃嘿嘿嘿，回来又吃了车仔面和一杯豆浆
  3. 和粥粥视频通话，两个人讨论去清远的计划，本来想去漂流，可惜了大概开不了门，后来想去温泉，觉得太偏了，最后打算就去清远吃东西了
  4. 晚上打游戏，有柴、我妹、粥粥、我还有一个路人，我上王者了，粥粥很生气
  5. 顺路取了快递

2024.10.30

  1. 早上去医院拿报告，没事了，医生说没问题，可以合格了，以后再去复查吧
  2. 又点了一份老北京鸡肉卷、车仔面、豆浆，不知道是什么原因，喉咙好干
  3. 粥粥睡午觉，我看看书，写写日记，真好
  4. 书里看到一个片段觉得不错，分享给粥粥，我：一家三口，还不错，粥粥：你又不错啦
