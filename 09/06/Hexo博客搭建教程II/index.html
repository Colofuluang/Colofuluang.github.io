Hexo 博客搭建教程 II

2024/9/6 教程 Hexo 主题

接上文： Hexo 博客搭建教程 I。

1. 安装主题

首先在这里选择一个主题。

上面的图片是预览页，点击标题就可以到达对应主题的 Github 项目页。

选择心仪的主题之后进行安装，在博客网站根目录下运行下面的命令将主题 Clone 到本地。

cd themes
git clone <link>.git <theme> --depth=1
// --depth=1 是为了 Clone 更快，只 Clone 最新提交。

我选择的主题是 ParticleX，如果你也喜欢这个主题，那么你只需要执行如下命令即可。

git clone https://github.com/theme-particlex/hexo-theme-particlex.git particlex --depth=1

安装完成后，在博客根目录下的 _config.yml 中设置 theme 参数为该主题名称，就能够实现主题的切换，一般主题在 Github 项目页下都有介绍和配置说明，可以按照说明自定义页面。

2. 创建特殊页面

2.1 分类页

执行如下命令，创建分类页面。

hexo new page categories

打开 source/categories/index.md，在 --- 括起来的地方添加 type : categories。

2.2 标签页

hexo new page tags

打开 source/tags/index.md，在 --- 括起来的地方添加 type : tags。

2.3 关于页

hexo new page about

打开 source/about/index.md，在下面添加内容即可。

3. 自定义网站配置

网站配置存放在博客根目录下的 _config.yml 文件中。

具体可以参考官方文档：Hexo文档
